Antivirus engine with a set of virus databases and regular updates. Detects and cures all types of malware. Allows you to create your own anti-virus software without the creation of a personal virus lab, connect antivirus functionality to applications.

Package contents
-----------------------------------------------------
app - Contains the source code of applications that are present in package
bin - The binaries of AvEngine and utility.
docs - Description and function of SDK
examples - Example of connecting the engine to the application
include - Header files needed connection engine
lib - Library for interacting with the service
